# Data Manipulation and Visualization Project - start2impact

This repository contains the Data Manipulation and Visualization project completed during the data science master at start2impact.

## Feedback: 500/500

## Project Overview

Analysis of the environmental impact of global food and feed production and distribution, exploring the relationship between food production, population growth, and climate change.

## Requirements

- Python 3.7+
- Jupyter Notebook or JupyterLab
- NumPy
- Pandas
- Seaborn
- Matplotlib
- Plotly
- country-converter
- raceplotly
- scikit-learn
- nbformat

## Installation

```bash
pip install jupyter numpy pandas seaborn matplotlib plotly country-converter raceplotly scikit-learn nbformat
```
