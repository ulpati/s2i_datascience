# Master in Data Science - start2impact

This repository contains all projects completed during the data science master program at start2impact.

## Projects

### [SQL (psql_travel)](https://github.com/ulpati/s2i_datascience/tree/master/psql_travel)
SQL project analyzing travel safety data from the U.S. Department of State.

### [Python and NumPy (file_organizer)](https://github.com/ulpati/s2i_datascience/tree/master/file_organizer)
File organization tool built with Python and NumPy.

### [Data Manipulation and Visualization (data_vis)](https://github.com/ulpati/s2i_datascience/tree/master/data_vis)
Data analysis and visualization project.

### [Machine Learning (machine_learning)](https://github.com/ulpati/s2i_datascience/tree/master/machine_learning)
Machine learning project applying predictive modeling techniques.

### [Final Project (final_project)](https://github.com/ulpati/s2i_datascience/tree/master/final_project)
Capstone project integrating all skills learned during the master program.
